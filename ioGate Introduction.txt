ioGate（客户端）
1、将ioGate文件夹放入mods文件夹下
2、使用Miko_Rouge.bat启动游戏后，确保在Mod选项中启用ioGate和它所依赖的Mods
3、在游戏主界面的左上角，使用LunaLib设置ioGate相关参数
4、使用ioGate建立连接时，确保本地或远程的ioHub已经完全启动

ioGate (Client)
1. Place the ioGate folder in the mods folder.
2. After launching the game using Miko_Rouge.bat, make sure to enable ioGate and the mods it depends on in the Mod Options.
3. In the upper left corner of the main game interface, use LunaLib to set the ioGate parameters.
4. Make sure the local or remote ioHub is fully started when you use ioGate to establish a connection.
